# Design summary

- Design concept: Helpdesk war room
- Layout personality: futuristic technology
- Typography: Rajdhani headings with Inter body text
- Color palette: primary #0f172a, secondary #22d3ee, accent #a3e635, background #020617, text #e2e8f0, muted #94a3b8
- Shape language: Sharp corners
- Visual motif: dark panels and alert lights
- Unique layout choices: first sections use Support war room, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
